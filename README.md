# Employee Salary Dashboard Mini Project (Excel)

## Overview

This project is an interactive Excel dashboard that analyzes employee salary distribution across different departments.

It provides quick insights using visual charts, KPI cards, and interactive filters.

---

## Features

* Pivot Table for data aggregation
* Pivot Chart for salary comparison
* KPI Cards:

  * Total Employees
  * Average Salary
  * Highest Salary
* Slicer for interactive filtering (Department)
* Clean and minimal dashboard design

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Basic formatting and UI design
